# Conectada

## Sobre
Esse é o aplicativo do projeto [Conectada](https://conectada.io).

## Como instalar
```
npm install -g yarn
npm install -g ionic
npm install -g cordova
yarn install
cordova platform add android
yarn android
```

## Scripts
Desenvolver
```
yarn serve
```

Build
```
yarn android
```
